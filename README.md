# Contact Manager Frontend

## Overview
This is the frontend application for the Contact Manager system. It provides a user interface for managing and organizing contacts.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Running the Application
```bash
npm start
```

The application will run on `http://localhost:3000`.

## Project Structure
```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── services/      # API service calls
├── styles/        # CSS/styling
└── App.js         # Main application component
```

## Features
- View and manage contacts
- Add new contacts
- Edit existing contacts
- Delete contacts
- Search and filter contacts

## Build
```bash
npm run build
```
